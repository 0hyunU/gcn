# Graph Convolutional Networks

Implementation of [Semi-Supervised Classification with Graph Convolutional Networks](http://arxiv.org/abs/1609.02907) in Tensorflow.

## Installation

```bash
python setup.py install
```

## Requirements
* tensorflow (>0.10)
* networkx

## Run

```bash
python train.py
```
